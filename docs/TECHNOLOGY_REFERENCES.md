# Technology References

This project relies on a mixture of scientific datasets, algorithms, and software libraries. The references below cover the primary technologies and data sources explicitly referenced in the codebase and documentation. All references are peer-reviewed publications, official technical standards, or authoritative sources with verifiable DOIs where available.
